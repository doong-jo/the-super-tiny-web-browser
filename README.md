## Intro
웹 브라우저의 학습목적으로 개발한 최소한의 브라우저 기능이 동작하는 웹 브라우저입니다. 

- URL validation
- Host에 **request** 그리고 **response** 처리
- 렌더링은 Chromium을 이용 (학습 참고자료 👉 [Naver D2 브라우저는 어떻게 동작하는가?](https://d2.naver.com/helloworld/59361))

## Usage
```bash
# 실행
npm install
npm start
```
